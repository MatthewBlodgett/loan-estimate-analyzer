# Loan Estimate Analyzer

A single-file HTML tool that decodes mortgage Loan Estimates into plain English. Built with 20 years of mortgage domain expertise.

## What It Does

Enter the numbers from your Closing Disclosure (CFPB Loan Estimate form) and instantly get:

- **10+ domain-expert checks** — rate vs market, APR spread, fee analysis, LTV, prepayment penalties, balloon payments, escrow review, and more
- **Scored rating** — 0–100% overall health grade with color-coded findings
- **AI-powered summary** — optional plain-English advisory letter via OpenRouter free models (bring your own free API key)

## Why It Exists

Loan Estimates are confusing by design. Most borrowers can't tell a good deal from a bad one. This tool translates regulatory forms into actionable advice — no license, no software install, just one HTML file.

## Tech

- **100% client-side** — no backend, no server costs, no tracking
- **Single HTML file** — open it, use it, share it
- **Stripe-inspired design** — clean fintech UI with universal fonts
- **OpenRouter API** — free-tier AI enhancement (optional)

## How to Use

1. Open the live site: https://matthewblodgett.github.io/loan-estimate-analyzer/
2. Fill in the fields from your Loan Estimate
3. Click **Analyze Loan Estimate**
4. For an AI-written advisory, click **Enhance with AI Analysis** and paste a free [OpenRouter](https://openrouter.ai/keys) API key
